About This Site

This website is a simple static template built with plain HTML and CSS. It features a fixed navigation bar at the top so you can move between pages without having to scroll back up.

The main content area automatically centers itself and adjusts to any screen size, ensuring your text and images look great on both desktop and mobile devices.

The products page includes a flexible image grid that adapts to however many photos you add. To customize the look, update the color values in css/style.css and drop your own images into the images/ folder.
